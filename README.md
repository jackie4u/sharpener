# CSharpener ⚔️

A curated collection of bite-sized, logic-driven coding challenges, mini-games, and practical syntax exercises. 

## 🎯 The Aim of this Repository

Sometimes coding just needs to be fun again. This repository serves as a personal sandbox to step away from heavy decisions, focusing instead on quick, dopamine hits. 

The core philosophy here is **logic and mechanics over heavy math**. You won't find deep algorithmic time-complexity whitepapers or abstract calculus here. Instead, these are practical, engaging, and often game-based puzzles designed to keep the syntax sharp, explore UI experiments in Blazor or HTML/CSS, and keep the mind engaged without the fatigue of a university exam.

## 🛠️ The Stack

* **Primary:** C# / .NET
* **Secondary:** TypeScript, JavaScript
* **Occasional:** UI experiments (HTML/CSS/Blazor) when visual feedback is needed.

## 🕹️ Puzzle Sources

The snippets in this repository are sourced from platforms that prioritize engaging formats over dry, abstract algorithms. 

* **[CodinGame](https://www.codingame.com/):** Instead of returning arrays or calculating primes, code here controls a spaceship, intercepts missiles, or navigates a maze. It provides high-quality visual feedback where you actually see a mini-game play out in the browser based on your C#, TypeScript, or JavaScript code. Single-player puzzles take about 15–30 minutes, and "Clash of Code" offers fast-paced, 15-minute multiplayer mini-battles that focus purely on quick logic rather than deep math.

* **[CheckiO](https://checkio.org/):** A sci-fi/fantasy world where you solve coding challenges in TypeScript to progress to different "islands" or stations. The solutions of other users are highly visible and discussed, offering a great community learning aspect that can easily be broken down into 15 to 30-minute chunks.

* **[Elevator Saga](https://play.elevatorsaga.com/):** A completely browser-based, pure logic game that is incredibly addictive. You are tasked with programming the movement logic of elevators using JavaScript. You start simple and slowly have to optimize for wait times, max capacity, and multiple shafts. It is incredibly practical state-management logic, completely devoid of abstract math, where a level or two can easily be tackled in 20 minutes.

* **[Edabit](https://edabit.com/):** If gaming visuals aren't strictly necessary, Edabit is a brilliant option designed entirely around practical language mechanics, not competitive programming. Tasks include formatting dates, parsing specific strings, or manipulating objects in C# and JavaScript. It is truly bite-sized, with many exercises taking less than 5 minutes.

* **[Codewars](https://www.codewars.com/):** Heavily curated to avoid algorithmic nightmares.
